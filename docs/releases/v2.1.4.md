# GrtBlog v2.1.4

- 发布日期（UTC）：2026-07-18
- 对比基线：v2.1.3
- 发布通道：Stable

v2.1.4 集中完善了内容分享与阅读体验，并修复了时间线图片和完整站点恢复后的渲染一致性问题。

## 分享与阅读体验

- 文章、手记和相册详情页新增统一的分享菜单，可选择系统分享或生成分享卡片。
- 分享卡片会根据当前内容生成标题、摘要、封面、站点信息、完整文章地址和二维码；生成前会等待字体、头像与封面资源加载完成。
- 移动端使用底部浮动窗口预览卡片，支持直接分享或保存图片；桌面端提供紧凑的预览与保存操作。
- Markdown 新增 `details` 折叠内容块，并在目录跳转到折叠区域内的标题时自动展开对应内容。

## 管理与内容展示

- 高级渲染详情中的路由目录改为树形结构，支持搜索、展开一级、全部展开和全部收起，并使用统一滚动容器承载长列表。
- 修复时间线手记包含多张逗号分隔图片时拼接出无效地址的问题；前后端现在都只取第一张图片作为时间线预览。
- 增加 Markdown AST 目录测试，确认代码块、表格、行内代码与 HTML 块中的井号不会被误识别为标题。

## 备份恢复与工程优化

- 完整站点恢复成功后会清理站点派生缓存和旧的 ISR Bootstrap 标记，服务重启时自动执行全站预渲染，避免恢复数据库后仍继续使用旧页面状态。
- 备份恢复容器端到端测试新增派生缓存清理与全站 Bootstrap 验证。
- CI 为 Go 依赖和备份 E2E 镜像增加缓存复用；CNB 镜像同步任务下调为单核配置，减少重复构建与发布资源消耗。

## 升级说明

- 本版本不包含新的数据库迁移，按现有升级流程更新到 `2.1.4` 即可。
