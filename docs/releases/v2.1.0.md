# GrtBlog v2.1.0

- 发布日期（UTC）：2026-07-13
- 对比基线：v2.0.4
- 发布通道：Stable

v2.1.0 是一次覆盖内容形态、站点互联、阅读体验、后台管理和部署链路的完整更新。除了新增相册与联合功能，本次版本还集中修复了 ISR 静态页面生成、联邦投递、内容互动和升级部署中的稳定性问题。

## 新增相册

- 后台新增相册管理，可创建相册、编辑 Markdown 描述、管理封面并批量维护照片。
- 前台新增相册列表、相册详情与独立照片查看页，完善了图片切换动效和移动端布局。
- 相册已接入浏览、点赞与评论，相关数字会在页面打开后刷新，评论也可实时更新。
- 上传文件可同步进媒体索引，便于将已有图片纳入统一管理。

## 联合与 ActivityPub

- 联合功能与 ActivityPub 已在后台开放，可在升级引导或「设置 → 联合」中按需启用。
- Mastodon、Misskey 等 Fediverse 平台的用户可以关注站点，并在时间线中接收新文章与手记。
- 改进远端文章引用与提及：编辑内容时可直接输入远端地址检索，不再要求双方已建立友链。
- 完善联合友链申请、审批、实例创建、即时同步和出站投递流程。
- ActivityPub Actor 支持配置头图，并优化推送内容在第三方客户端中的展示。
- 加固远端请求校验、投递幂等、重复任务处理和外部 URL 安全，减少重复投递与异常同步。

## 阅读与互动体验

- 文章、手记、页面、思考和相册的浏览、点赞、评论数字会在客户端刷新，降低静态缓存带来的数据延迟。
- 详情页数字变化加入滚动动画；新评论可通过实时连接自动刷新，无需手动重新加载页面。
- 时间线支持触控板和触屏横向手势，并修复手势冲突与误触问题。
- 未填写摘要时，会从 Markdown 正文提取更干净的首段内容。
- 修复思考浏览量重复统计、手记短链接日期未使用站点时区等问题。

## 后台与运维

- 新增任务式升级引导，升级后可集中确认并启用本版本引入的可选功能。
- 重做上传管理体验，支持图片/文件筛选、上传进度、索引同步、预览、重命名、下载与删除。
- 新增内容导出脚本，可将文章、手记、思考和独立页面导出为 Markdown、元数据与清单文件。
- 保存系统配置后会保留当前筛选状态，并优化统一设置面板和编辑器的交互。
- 修复导航拖拽排序、ActivityPub 评论头像、友链申请说明等问题，并加入过期分析数据、AI 日志和邮件记录的定时清理。

## 隐私与安全

- 新增完全可选的匿名遥测。该功能默认关闭，新安装时需由站点管理员明确选择；已有站点升级后仍保持关闭，可在设置中手动开启。
- 管理员可在「设置 → 帮助我们变得更好」中预览待上报数据、修改上报端点与间隔，或随时关闭遥测。
- 遥测不会发送账号凭据、正文内容、访客身份、浏览轨迹或 IP 地址。
- 加强评论、点赞和联合接口的限流与客户端识别，并完善 AI 内容审核和提示词防护。

## 静态优先与稳定性

- 修复 ISR 与 Renderer 的协作问题，恢复静态文件优先的访问路径；后端或渲染器短暂不可用时，已生成页面仍可继续提供只读访问。
- 增加 ISR 版本标记、资源清单和全量重渲机制，减少升级后页面与资源版本不一致的问题。
- 完善渲染错误日志、错误页面和分页路由，便于定位生成失败并避免异常页面覆盖有效快照。
- 更新前后端依赖和构建链路，并补充联合功能的双实例端到端测试。

## 升级须知

推荐重新运行一键安装脚本完成升级。脚本会备份并同步部署配置、更新版本号、拉取镜像、重启服务并重载 Nginx：

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/grtsinry43/grtblog/main/deploy/install.sh)
```

国内网络可使用：

```bash
bash <(curl -fsSL https://cnb.cool/grtsinry43/grtblog/-/git/raw/main/deploy/install.sh)
```

手动升级时请特别注意：

- 必须同步本版本的 `docker-compose.yml` 与 `nginx/nginx.conf`，不能只替换 Server 和 Renderer 镜像。
- 本版本新增 `./storage/meta` 共享目录，用于保存 ISR 版本标记与资源清单；缺少该挂载会影响静态页面更新。
- `nginx/nginx.conf` 现在由 GrtBlog 部署配置管理。自定义反向代理规则建议放在外层代理中。
- 数据库迁移会在 Server 启动时自动执行；升级后还会进行一次静态页面全量重渲，首次启动可能比平时更久。
- 匿名遥测默认关闭，已有站点升级后不会静默开启。

完整的手动升级步骤请参阅[部署文档](/guide/deployment)。
