# 个性化配置

GrtBlog 提供丰富的配置选项，让你可以轻松定制博客的外观和行为。大部分配置可以直接在管理后台完成。

## 站点基本信息

在管理后台「系统配置」中设置：

- **站点名称** - 显示在浏览器标签和页头
- **站点描述** - 用于 SEO 和社交分享
- **站点图标** - Favicon 和社交分享图
- **作者信息** - 姓名、头像、简介

## 评论系统

GrtBlog 内置评论系统，支持：

- 游客评论 / 登录评论
- Cloudflare Turnstile 人机验证（防垃圾评论）
- 评论审核与管理
- 邮件通知

在管理后台「系统配置」中开启或关闭评论相关功能。

## 联合社交

GrtBlog 的社交互联基于**自有联合协议**，这是核心的社交能力。在管理后台「联合」模块中配置：

- GrtBlog 实例之间的内容分发与订阅
- 联合协议的配置与联动管理

### ActivityPub 兼容

作为补充，GrtBlog 兼容 ActivityPub 协议，在管理后台「联邦」模块中配置：

- Mastodon / Misskey 用户可以搜索并关注你的博客
- 新文章发布时自动推送到 Fediverse 关注者的时间线
- 来自 Fediverse 的回复会显示为评论

::: tip
联合是 GrtBlog 的核心社交协议，ActivityPub 是兼容层。两者在管理后台中分别独立配置。
:::

## 邮件通知

配置邮件服务后，系统可以发送：

- 新评论通知
- 评论回复通知
- 友链申请通知

在管理后台「邮件设置」中配置 SMTP 信息。

## Webhook

支持配置 Webhook，在特定事件（如新文章发布、新评论等）发生时向外部服务发送通知。

## 搜索

GrtBlog 集成了全文搜索功能，读者可以快速搜索文章内容。搜索由后端提供支持，无需额外配置。

## 数据统计

管理后台「数据统计」提供：

- 访客统计和趋势
- 文章阅读量排行
- 访客地理分布（需启用 GeoIP）
- 实时在线人数
