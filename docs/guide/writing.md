# 写作指南

GrtBlog 提供多种内容形式，满足不同的创作场景。所有内容均在管理后台撰写和管理。

## 进入管理后台

访问 `https://your-domain/admin/`，使用管理员账号登录。

## 文章

文章是博客的主要内容形式，支持完整的 Markdown 语法。

### 创建文章

1. 进入管理后台，点击左侧菜单「文章管理」
2. 点击「新建文章」
3. 填写标题、摘要、正文
4. 设置分类和标签
5. 点击「发布」

### Markdown 语法

编辑器支持标准 Markdown 语法，并提供实时预览：

- 标题：`# H1` ~ `###### H6`
- 粗体：`**文本**`
- 斜体：`*文本*`
- 链接：`[文字](URL)`
- 图片：`![描述](URL)`
- 代码块：使用三个反引号包裹，支持语法高亮
- 表格、引用、列表等标准语法

### 自定义组件

除了标准 Markdown，GrtBlog 还支持自定义组件来丰富内容表现力：

- **相册 (Gallery)** - 在文章中嵌入图片集
- **卡片 (Card)** - 展示链接卡片

::: tip
自定义组件在管理后台编辑器中可以直接插入，无需记忆语法。
:::

## 手记 (Moments)

手记的功能与文章基本一致（支持完整 Markdown、目录导航、关联推荐等），但定位更偏向生活记录和日常分享，适合：

- 生活随笔和日常记录
- 旅行见闻和体验分享
- 不那么"正式"的个人内容

在管理后台「手记管理」中创建和管理。

## 思考 (Thinking)

思考是轻量级的短内容形式，适合：

- 随手记录的灵感碎片
- 一闪而过的想法
- 简短的观点和感想

在管理后台「思考管理」中创建和管理。

## 页面管理

除了文章、手记和思考，你还可以创建自定义页面，例如「关于我」、「项目展示」等。

## 友链管理

在「友链管理」中添加和维护你的博客朋友圈。友链页面支持 SSR，对 SEO 友好。

## 导航菜单

通过「导航管理」自定义网站的导航菜单，支持多级菜单和图标配置。

## RSS 订阅

系统自动生成 RSS 订阅源，读者可以通过 RSS 阅读器订阅你的博客更新。RSS 相关设置在「RSS 管理」中配置。
