# 什么是 GrtBlog？

GrtBlog 是一个现代化的博客系统，专为追求极致阅读体验和写作效率的创作者设计。

## 核心特性

### 极速加载

页面以纯静态 HTML 形式分发，首屏加载时间通常低于 0.5 秒。无论你的读者身在何处，都能获得接近本地打开的速度体验。

### 实时更新

当你修改文章内容后，正在阅读的读者无需刷新页面即可看到最新内容。评论、点赞等互动数据也会实时同步。

### 丰富的内容形式

- **文章** - 正式的长文内容，支持完整的 Markdown 语法、代码高亮、数学公式、自定义组件，接入全部推荐系统
- **手记 (Moments)** - 与文章功能相近，但更偏向生活记录和日常分享
- **思考 (Thinking)** - 轻量级的短内容，适合记录灵感碎片和随想
- **友链** - 展示你的博客朋友圈
- **时间线** - 按时间线浏览所有内容

### 联合社交

GrtBlog 拥有**自有的联合协议**，支持 GrtBlog 实例之间以及与其他平台之间的内容分发、订阅和互动，让你的博客不再是一座孤岛。

此外，GrtBlog 也兼容 ActivityPub 协议，Mastodon、Misskey 等 Fediverse 平台的用户同样可以关注你的博客、接收更新。

### 功能管理后台

提供一个美观且功能完备的管理后台，支持：

- Markdown 实时预览编辑器
- 文章分类与标签管理
- 评论审核与管理
- 访客数据统计
- 系统配置与主题设置
- RSS 订阅管理

## 技术亮点

| 特性 | 说明 |
|------|------|
| 首屏速度 | < 0.5s（静态文件直出） |
| 更新延迟 | WebSocket 推送 < 100ms |
| 内存占用 | 闲置状态 < 100MB |
| 安全评级 | A+（SSL, HSTS, CSP） |

## 适合谁？

- 希望拥有独立博客、掌握数据主权的创作者
- 对网站性能有较高要求的技术博主
- 希望博客能与其他平台互联互通的内容创作者
- 需要一个支持多种内容形式的个人站点
